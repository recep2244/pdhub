# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Protein Design Hub, please report it privately.

**DO NOT** open a public issue on GitHub.

Instead, please email us at:
`security@example.com` (Replace with actual email)

We will verify the issue and respond within 48 hours with a timeline for fixing it.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |
